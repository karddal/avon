# Frontend setup

The frontend runs on Next.js. This means that you will need a Typescript environment set up. Follow these instructions to get everything set up:

- Install a node version manager e.g. `nvm`, so that you have access to `npm` and `node`.
- Run `just sync` to download and install dependencies
- Ensure you have [set up the environment variables](environment_variables.md)
- Run `just run-fe` to start up the frontend
