# Making issues

The first step of any contribution is the creation of an issue on the [Main Kanban](https://github.com/orgs/spe-uob/projects/309). You should use the templates provided and provide details about your feature, why it should be included. The same goes for bugs as well. You should use the labels to make sure that it is properly categorised.

If you are going to be working on the issue immediately, you should assign it to yourself, so that nobody else starts working on it.
