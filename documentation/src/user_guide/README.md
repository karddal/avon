# User instructions

## Students
### Logging in
- Your account would have already been made for you. To log in, go to the login page (/login) and put in your Bristol University email and password.
### Viewing Units
-  Once logged in, you can now view all the units (/units) you are taking. You can filter these by either the 'programme' (e.g. Year 1 Computer Science) or whether they are ongoing or finished.
-  If you can click on a specific unit you will be able to view important details about the unit. This includes the unit description, staff, announcements and most importantly, coursework.
### Viewing Coursework
- You can either view a specific coursework by clicking on them via the unit page or, you can navigate to /courseworks and view all the courseworks. Similar to the Units page, you can filter the courseworks by Unit or whether they are ongoing or finished.
- Once you click on a specific coursework, you can see important information about the coursework. Along with the description and important information, you can see your activity (the number of tests you've passed so far). There is also a button to run tests. Although there are no actual tests running at the moment, clicking the button will deliver a success message.
### Switching between light and dark mode
- You can switch between dark mode and light mode by clicking on the sun/moon icon at the top right of the page.
### Viewing Announcement
- You can view announcements by clicking on the top right bell icon.
### Loggin out
- You can logout by clicking on the User icon with your name on the bottom left. Click the icon and press logout to be taken back to the login page (/login)

## Lecturers

- Enter your username and password and select `Log in`.

### Viewing units

- Select the units link on the sidebar.

### Setting coursework

- Select your unit from the units list.
- In the courseworks section on the unit page, select the `Assign new coursework` button.
- Fill in the details needed in the form, including due date, colour, name, description.

### Viewing coursework

- Select the coursework section in the sidebar.
- You can see the list.

### Editing coursework

- Go to the specific coursework page. You can get here from the QuickJump `Ctrl+K`, through the unit page, or from the courseworks listing.
- Click on the dropdown menu in the top right of the screen.
- Select `Edit coursework`.
- Change anything you'd like in the form that appears.
- Select `Save` to finish the process.
